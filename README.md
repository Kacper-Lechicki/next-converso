# Next Converso

An intelligent AI mentorship platform connecting users with virtual companions for real-time interactive sessions and personalized learning journeys.

### Key Capabilities

- **AI Companions:** Engage with specialized virtual mentors powered by Vapi AI.
- **Real-time Interaction:** Seamless voice and text-based communication.
- **Session Management:** Track progress with detailed session history and transcripts.
- **Secure Access:** Enterprise-grade authentication via Clerk.
- **Global Reach:** Multi-language support built-in.

### Tech Stack

- **Core:** Next.js 16, React 19, TypeScript
- **AI & Data:** Vapi AI, Supabase
- **Auth:** Clerk
- **Styling:** Tailwind CSS 4, Framer Motion, Radix UI

### Getting Started

```bash
git clone https://github.com/Kacper-Lechicki/next-converso.git
cd next-converso
npm install
cp .env.example .env.local
npm run dev
```
